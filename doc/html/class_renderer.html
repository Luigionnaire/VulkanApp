◆ createSyncObjects()

void Renderer::createSyncObjects ( )
private

Create synchronization objects (semaphores and fences) used for rendering frames.

Creates MAX_FRAMES_IN_FLIGHT semaphores for image availability and render completion, and fences to ensure CPU-GPU synchronization. Throws runtime_error if creation fails.

◆ drawFrame()

void Renderer::drawFrame ( )
private

Render a single frame.

Waits for previous frame to finish, acquires next swapchain image, updates uniform buffers, records command buffers, submits draw commands, and presents the rendered image. Handles window resizing and swapchain recreation.

void Renderer::recordCommandBuffer ( VkCommandBuffer commandBuffer,
uint32_t imageIndex )
private

Record commands into a command buffer for rendering a frame.

Records commands to begin render pass, bind pipeline, set viewport and scissor, bind buffers and descriptor sets, draw the model, and end the render pass.

Parameters
commandBufferThe Vulkan command buffer to record commands into.
imageIndexThe index of the swapchain image to render to.
Exceptions
std::runtime_errorif command buffer recording fails.
